String.prototype.toLowerCase()

The toLowerCase() method returns the calling string value converted to lower case.

Syntax

str.toLowerCase()

Return value:

A new string representing the calling string converted to lower case.

Description

The toLowerCase() method returns the value of the string converted to lower case. toLowerCase() does not affect the value of the string str itself.

W3Schools

The toLowerCase() method converts a string to lowercase letters.
Note: The toLowerCase() method does not change the original string.
Tip: Use the toUpperCase() method to convert a string to uppercase letters.
